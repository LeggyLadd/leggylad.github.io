Assignment 14 - Flappy Tarnished

HOW TO PLAY:

To start playing, simply press the start button. The Tarnished, riding a physics-defying horse, has to avoid all the obstacles on their endless journey to become the Elden Lord. You can help him get there by left-clicking the mouse within the game in order to make the horse bounce. You will receive a point each time the player passes an obstacle, and the game will tell you how well you did at the end of the each run. Good luck!
