BSN Quick Class Advisor

This quick advisor allows for quick planning of course schedules.

To use the planner, first, check all courses that you have previously completed. If you have taken a course equivalent to a specific one, you should check that course. Once you have completed checking all courses you have taken, you can plan courses directly by clicking on a course in any semester or use the automated tools to plan a draft schedule. You can create a printable schedule after completing your plans. This tool is meant to be a preliminary tool to help students in planning, and students should still seek regular advising.
